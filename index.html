Configurar projeto Angular CLI + TypeScript

  1. npm install -g @angular/cli
  2. ng --version
  3. New directory: ng new my-app
    or
    Existing Directory: ng new appName --directory ./
  4. Em caso de erro com o NPM:
    • npm cache verify
  5. Plugins para Visual Studio Code:
    • https://marketplace.visualstudio.com/items?itemName=EditorConfig.EditorConfig
    • https://marketplace.visualstudio.com/items?itemName=johnpapa.angular-essentials
  6. Plug-in para o Chrome:
    • https://augury.rangle.io/
  7. ng serve --open
  8. Codar o projeto!
  9. Done!!!
